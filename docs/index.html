Activity

کد های برنامه از کلاس های Activity یا Application شروع به اجرا شده میکنند. هر Activity در واقع یکی از صفحه های اپلیکیشن است. هر برنامه اندروید باید حداقل یک Activity داشته باشد.

Services

المانی از اندروید هستند که ظاهر گرافیکی ندارند و صرفا وظیفه اجرای منطق برنامه را دارند. Service ها دو نوع اند.
Background
Foreground

Broadcast Receiver

مکانیزمی برای انتقال پیام هستند. گاهی اوقات سیستم عامل اندروید پیام هایی را به برنامه میدهد. گاهی اوقات هم برنامه های دیگر نیاز است با هم پیام هایی را رد و بدل کنند. برای این کار از Broadcast receiver استفاده میشود. اینها المان هایی هستند که به پیام های خاصی گوش میدهند.

Manifest

Manifest یا مرام نامه اطلاعات کلی برنامه را نگهداری میکند
